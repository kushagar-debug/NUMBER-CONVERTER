# Number System Converter

## Description
This is a simple C program that allows users to convert decimal numbers into different number systems â€” Binary, Octal, and Hexadecimal. It features a menu-based interface and includes input validation to prevent infinite loops on invalid input.

## Features
- Convert Decimal to Binary
- Convert Decimal to Octal
- Convert Decimal to Hexadecimal

## How to Run
1. Compile the code:
```

```


2. Run the program:
```

```
